# Getting Started
Spring Boot + Kotlin + Akka Streams

## Endpoints
Test Ping 
```
curl -X GET http://127.0.0.1:8080/feeds/ping
```

Get All
```
curl -X GET http://127.0.0.1:8080/feeds
```

GET User
```
curl -X GET http://127.0.0.1:8080/feeds/{user_id}
```

DELETE User
```
curl -X DELETE http://127.0.0.1:8080/feeds/345
```
