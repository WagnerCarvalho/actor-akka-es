# akka-es-example
> A showcase application using the [akka] structure (https://akka.io/ "akka"), to emulate social feeds guaranteeing the idempotency, parallelism and simultaneity of the events.

## Stack
*   Akka
*   Akka-Http
*   SpringBoot
*   Kotlin
*   ElasticSearch
